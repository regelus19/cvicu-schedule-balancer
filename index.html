🏥 CVICU Schedule Balancer — What This Tool Does
This tool takes a MyTime (UKG/Kronos) schedule export and automatically fills in coverage gaps for both Day and Night shifts — respecting all CVICU scheduling rules, staff type requirements, weekend obligations, and shift separation. It never modifies ANMs and preserves self-scheduled shifts wherever possible.
💻 About This File — Sharing & Browser Requirements
📁 Completely Standalone
This is a single .html file. Everything — the balancing logic, staff roster, all scheduling rules — is built into the file itself. No login, no account, no subscription required.
🌐 Works in Any Modern Browser
Open the file in Google Chrome, Microsoft Edge, Safari, or Firefox. Simply double-click the file or drag it into a browser window. Chrome or Edge recommended for best experience.
📤 How to Share
Send the CVICU_Balancer_v30.html file as an email attachment, via Teams, or upload to SharePoint. Recipients just open it in their browser — no installation needed.
🔒 Data Stays on Your Computer
Your schedule data is never uploaded to any server. All processing happens locally in your browser. Nothing leaves your machine.
📶 Internet Connection
An internet connection is only needed for the Excel download feature (loads the Excel library from a public CDN). All other features — upload, balance, view results — work fully offline.
💾 Nothing is Saved Between Sessions
The tool uses your browser's local storage to remember the Master Staff Database between sessions on the same device. Clearing browser data will reset it. Always export the master CSV as a backup.
📤 Step 1 — Upload MyTime Export
  1. In MyTime/UKG, run the Employee Schedule Monthly report
  2. Export as .xlsx
  3. Drag & drop the file onto the upload zone, or click to browse
  4. The tool reads all staff, dates, and shift codes automatically
✅ Supports 6-week pay period exports
⚙️ Step 2 — Set Coverage Requirements
  • RN Weekday Min — target RN count Mon–Fri (default: 15)
  • RN Weekend Min — target RN count Sat–Sun (default: 14)
  • Cap sliders — max RNs above minimum per day type:
      Sat +1, Sun +2, Mon/Tue/Wed/Fri +3, Thu +4
  • PCT/HUC Weekday min 2, PCT/HUC Weekend min 1 (PCT and HUC are counted together toward coverage)
  • CN (Charge Nurse) min 3 per shift
  • EB and ES minimums (skill mix)
👥 Step 3 — Review Staff Roster
  • Verify each staff member's Type (FT/PT/PRN)
  • Verify their Shift (Day AM / Night PM / Mid)
  • Verify their Skills (CN, EB, ES)
  • Use 💾 Master Staff Database to save this info permanently — future uploads will auto-fill it
  • ANMs (Cruz, Rodriguez, Bruce) are marked 🔒 and will never be moved
⚠️ Always verify shifts for new staff before balancing
🔒 Step 4 (Optional) — Protect Staff
  • Check any staff whose schedule must not be changed
  • Use for: school commitments, pre-approved vacation, FMLA, medical accommodations
  • Protected staff are skipped entirely during balancing
⚡ Step 5 — Run Balancing
  • Click 🔍 Analyze Schedule first to preview coverage gaps before making any changes — this button appears in the balancing panel once your file is loaded
  • ☀️ Balance Day Shift — fills AM + MID shift gaps
  • 🌙 Balance Night Shift — fills PM shift gaps
  • 🔄 Balance Both — runs day then night in one click
  • The algorithm: fills PCT/HUC first → CN → EB/ES → RN; respects all caps
  • Weekends are prioritized first in the fill order
💡 If results aren't ideal, adjust cap sliders and re-run
📥 Step 6 — Download Results
  • 📥 Download Excel — multi-sheet workbook:
      • Coverage Summary (weekday/weekend view)
      • Day Shift (ANMs → MID RNs → RNs → PCTs → HUCs)
      • Night Shift
      • Original MyTime Import (pre-balance)
  • 📋 Download Move Report — CSV of every change made
  • Review the Move Tracker section to see Added vs Moved per staff
⚡ Key Rules to Know
Shift Separation
Day (AM) and Night (PM) staff are completely separate pools. The tool never crosses them over.
Mid Shift (MID)
Counts toward Day coverage. Min 1 MID on Sat and Sun. Kesbeh & Hickman are MID.
PTO = Worked Shift
PTO, FMLA, LOA count toward the 18-shift (FT) / 12-shift (PT) minimum. SICK does not.
Weekend (Night)
Friday counts as a weekend day for night shift staff. Fri/Sat/Sun = 7 required.
Late Submitters
FT <18 or PT <12 shifts submitted = classified Late → gets unlimited moves assigned.
Move Caps
Compliant staff: max 2 changes. Non-compliant: max 6. Blank fills are free (not counted as moves).
PAYRO Code
ANM payroll admin code (4am–12pm office shift). Counts as worked, shows as PAYRO not PTO.
RN Cap (no +5)
Max overage: Sat +1, Sun +2, Mon/Tue/Wed/Fri +3, Thu +4. Hard ceiling enforced.
💡 Tips & Troubleshooting
First time setup: After loading your first MyTime file and verifying the roster, click 💾 Save Current as Master. Every future upload will auto-fill Job/Type/Shift/Skills from that saved database.
Staff showing wrong shift type? Edit their Shift dropdown in Step 3 and save as master. The Job field in MyTime must contain "(AM)", "(PM)", or "(MIDSHIFT)" for auto-detection.
Coverage still showing gaps after balancing? Some gaps can't be filled if all eligible staff have hit their weekly cap (3 shifts/week FT, 2 shifts/week PT). This is a genuine staffing shortage that requires agency or overtime.
Move tracker shows too many added shifts for one person? Always click Balance fresh — don't click the button multiple times in a row without reloading, as trackers reset on each run.
ANM schedule looks different from MyTime? ANMs are never moved. If their schedule looks off, re-export MyTime with a fresh file — the issue is in the source data, not the tool.
